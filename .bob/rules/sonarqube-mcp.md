# SonarQube MCP

Use the official `sonarqube` MCP server when inspecting code quality, security
hotspots, quality gates, duplications, coverage, and SonarQube issue context.

This template exposes the local SonarQube MCP endpoint at `SONAR_MCP_URL` in
`.env`. The MCP token is generated by `scripts/sonar-bootstrap.sh` and stored as
`SONAR_MCP_USER_TOKEN` in `.env.sonar`. Prefer SonarQube MCP findings over
guessing from static files when analysis results are available.
